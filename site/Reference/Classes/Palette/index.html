Palette

This module provides color palettes mapping colors and names to indices.

Colors can be specified as:

  • The name of a global color (e.g., black, white, grey-50, transparent).

  • The name of a color in a standard palette (e.g., c64.red, spectrum.bright-blue).

  • An integer in the format 0xAARRGGBB (where A value 0 represents opaque).

  • A tuple of RGB(A) values (where A value 0 represents transparent).

Returned color values are in the format 0xAARRGGBB.

c64 module-attribute

Palette for the Commodore 64 color palette.

global_colors module-attribute

Dictionary of global colors.

palettes module-attribute

Dictionary of standard palettes.

spectrum module-attribute

Palette for the ZX Spectrum color palette.

Palette(colors, names=None)

A palette mapping colors and names to indices.

PARAMETER DESCRIPTION
colors

A list or dictionary of colors mapping color values to indices.

TYPE: dict[int | str, int | None] | list[int | str | list[int | str]]

names

A list or dictionary mapping names to indices.

TYPE: dict[str, int] | list[str] | None DEFAULT: None
